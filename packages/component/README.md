# `@react-scan/component`

## Install

```bash
npm i react-scan @react-scan/component
```

```bash
pnpm add react-scan @react-scan/component
```

## Usage

```jsx
import { ReactScan } from "@react-scan/component";

export default function App() {
  return (
    <>
      <ReactScan enabled />
    </>
  );
}
```
